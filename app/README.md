# Run Instructions

## Development

```bash
bun run dev
```
Starts the development server with hot reloading.

## Build

```bash
bun run build
```
Builds the project for production.  
**Required files for production**:
- `/dist`
- `/server.ts`
- `/package.json`

## Production Server

```bash
bun run prod
```
Starts the production server using the built files.
